def reassign_level(inst, old_level, new_level):
    dz  = old_level.Elevation - new_level.Elevation        # ΔZ
    inst.LookupParameter("Reference Level").Set(new_level.Id)
    off = inst.get_Parameter(BuiltInParameter.INSTANCE_FREE_HOST_OFFSET_PARAM)
    off.Set(off.AsDouble() + dz)                           # Offset_new = Offset_old + ΔZ

Ghost Purge

Finds overlapping duplicates, keeps the connected copy and deletes the floating ghost — grouping by category/type first so it stays O(n) per group, not O(n²). Cleans massive models in seconds.

Connection Fix

Seals open MEP connector pairs within tolerance via spatial-grid bucketing, isolating each attempt in its own SubTransaction so one bad pair never rolls back the run.

Nested Align new

Catches co-located twins of the same family that sit at the same point but a different rotation — which Ghost Purge misses because their GetTransform() basis differs. Re-orients the higher-Id copy to match its anchor, turning a hidden near-duplicate into a detectable one.

Nested Resolve new

Finds host families with coincident nested sub-components whose visibility is driven by Yes/No parameters. Where a per-family rule says which one wins, it switches the redundant one off — clearing duplicates that live inside a single host.

Never acts on stale data. Every scan cache is keyed to model state and invalidated by any DocumentChanged event — reopening prompts a fresh rescan instead of trusting old rows. A Dashboard with quick-launch tiles, a JSON config page, CSV export and one-click Undo round out the workflow.

  • Automated BOQ Engine: Cross-references multiple element parameters against a master database (boq_data.py) to populate SPEC_DESIGNATION (Order ID) and SPEC_DESCRIPTION. Eliminates hours of manual referencing and instantly produces accurate Bills of Materials.
  • Smart Size Parsing: Reads messy inch strings and fractions (e.g. 1 1/2") and resolves them to metric Nominal Widths, with a fallback to Revit's native calculated size. Handles the real-world mess in supplier data.
boq_engine.py — automated size resolution
def lookup_boq(element, doc, read_param_func):
    size_str = read_param_func(element, "SPEC_SIZE", doc)

    # Fallback to Revit native size if SPEC_SIZE is empty
    if not size_str:
        p = element.get_Parameter(DB.BuiltInParameter.RBS_CALCULATED_SIZE)
        if p: size_str = p.AsString()

    # Parse complex inch strings into metric Nominal Widths (NW)
    if size_str:
        all_nws = []
        for sp in re.split(r'[-x×](?!\d/)', size_str):
            if '"' in sp:                                       # inch + fraction
                clean = sp.replace('"', '').strip()
                total = sum(float(f.split('/')[0]) / float(f.split('/')[1])
                            if '/' in f else float(f) for f in clean.split())
                all_nws.append(int(INCH_TO_NW.get(total) or round(total * 25.4)))
            else:                                               # already metric
                found = re.findall(r'\d+\.?\d*', sp.replace(",", "."))
                if found: all_nws.append(int(round(float(found[0]))))

        if all_nws:
            res = boq_data.BOQ_SIZE_INDEX.get((mat_seg, fit_seg, str(max(all_nws))))
            if res: return res                                  # → DESIGNATION, DESCRIPTION

    return (None, "Size '{}' not found in BOQ Index".format(size_str))

Shared Library icz/

Battle-tested utilities for pyRevit & Revit API add-ins. Write it once, reuse it everywhere.
Revit APIIronPython 2.7WPF / XAML Dependency-FreeRevit 2022–26 Safe

Every pyRevit tool in the suite solves the same fundamental problems: the Revit 2024 ElementId breaking change, modeless-window threading, theme duplication, worksharing guards, and MEP connector hashing. The icz module pulls all of it into one dependency-free library, so each tool imports instead of reimplementing.

revit_compat

The ElementId 32-bit ⇄ 64-bit shim for Revit 2024+. The one change every other module here is built to survive. Exposes eid(id) and make_eid(int).

mep

O(n) spatial-hash connector engine. Finds open pipe/duct connections within tolerance and reconnects them inside safe isolated SubTransaction blocks.

modeless

Wraps IExternalEventHandler into a reusable ExternalCall pattern. Ensures WPF button clicks safely mutate the model without crashing the UI thread.

theme & shell

One truth for Light/Dark palettes pushing into WPF {DynamicResource} properties, plus shared window chrome (traffic lights, collapsable nav-rail).

review_grid

Builds the shared in-window list rows (checkbox · ID · Family & Type · pill badges) used universally across tools for pre-scan reviews with click-to-isolate hooks.

mappings & params

Data-driven classification loaded from spec_mappings.json, alongside robust param-read functions handling Instance-then-Type fallback chains.

from icz import theme
from icz.revit_compat import eid, make_eid
from icz.modeless import ExternalCall
from icz.shell import ShellWindow

class ToolWindow(ShellWindow):
    def __init__(self, xaml):
        ShellWindow.__init__(self, xaml)
        self.init_shell(is_dark=True, rail_width=232)  # Shared chrome
        
        # Valid API context on init, safe threading
        self._mutate   = ExternalCall(self._commit)
        self._navigate = ExternalCall(self._zoom)

    def key_for(self, element):
        return eid(element.Id)                         # Int32 / Int64 safe
