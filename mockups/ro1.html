How the BOQ Classifier Works

◐ Step 1 · Material Segment

Reads SPEC_MATERIAL_TEXT_EN → looks up MATERIAL_TO_SEGMENT for the two-part material code (e.g. xx.xx. = stainless XXX). Exact match, then substring fallback. 95 materials mapped.

◑ Step 2 · Fitting Segment

Reads SPEC_NAME_EN → (material, name) in FITTING_TO_SEGMENT for the three-part fitting code. Family & Type resolves too — fallback when the name is absent, cross-check otherwise (name wins; a disagreement is flagged). 144 + 61 rules.

◒ Step 3 · Size Leaf (NW)

Reads SPEC_SIZE (or RBS_CALCULATED_SIZE) → parses NW (imperial fractions, DN, multi-port) → (mat, fit, NW) in BOQ_SIZE_INDEX → the atomic number + description. 672 sized leaves.
